浏览器解析HTML的过程

Posted on 2018-08-16 | In JavaScript

目标

在页面第一次渲染时就将样式与首屏有效内容呈现给用户。

页面的首次渲染发生时机

HTML中首个外部的js资源执行完，否则发生在HTML解析完毕；

现象

  • 包含外部资源的script标签置于body标签之前，外部js执行完之前不会进行首次页面的渲染；
  • 包含外部资源的script标签置于body标签之后，外部js执行完之前首次页面的渲染已经完成；
  • 包含外部资源的css标签置于body标签之前，外部css解析完之前不会进行首次页面的渲染；
  • 包含外部资源的css标签置于body与script标签之后，外部css解析完之前会进行首次页面的渲染，在css加载完成后会发生闪屏现象；

在server端通过增加资源的响应时间，便于肉眼观察资源的加载执行时间与页面呈现的先后顺序。
