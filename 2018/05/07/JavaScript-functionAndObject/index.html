函数与new对象的创建与执行

函数在创建时其环境已经给定了，其环境对象包含两个属性：函数体代码与一个指针指向函数创建时的外部环境对象

当函数接受参数被调用执行时，其执行过程如下：

  • 创建一个新的框架
  • 将参数进行处理并绑定到这个新的框架中
  • 在构造出的框架的环境的上下文中执行函数体中代码
  • 这个新的框架拥有独自封闭的环境，这个封闭环境是函数执行的环境的一部分

new 对象的执行过程

当使用new创建JavaScript对象时，需要一个指定其名称与属性的函数。对象的属性可以指定其他对象，例如，当代码new Foo(...)执行时：

  • 一个继承自 Foo.prototype 的新对象被创建。
  • 使用指定的参数调用构造函数 Foo ，并将 this 绑定到新创建的对象。new Foo 等同于 new Foo()，也就是没有指定参数列表，Foo 不带任何参数调用的情况。
  • 由构造函数返回的对象就是 new 表达式的结果。如果构造函数没有显式返回一个对象，则使用步骤1创建的对象。（一般情况下，构造函数不返回值，但是用户可以选择主动返回对象，来覆盖正常的对象创建步骤）
